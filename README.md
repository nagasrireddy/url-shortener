# 🔗 URL Shortener

A simple and efficient URL shortener built with **Flask** and **SQLite**, supporting analytics and custom shortened links. Deployed live using **Render**.

## ✨ Features

- 🔗 Shorten long URLs with a single click
- 📋 Copy shortened links easily
- 📊 Track click analytics for each shortened URL
- ✅ Modern, responsive UI
- 🔐 SQLite backend with Flask-SQLAlchemy
- 🐳 Deployed on Render

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS (Custom styling)
- **Backend:** Python, Flask
- **Database:** SQLite
- **Deployment:** Render
